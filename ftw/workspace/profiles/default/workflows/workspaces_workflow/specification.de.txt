[Workspaces Workflow (Verzeichnis mit Workspaces)]

Beschreibung: Dieser Workflow gilt für den Ordner, welcher alle Arbeitsräume enthält.


Initialzustand: Offen


Rollen Zuweisung:
  Besitzer => Owner
  Benutzer => Authenticated
  Arbeitsraum Ersteller => Reviewer
  Teammitglied => Contributor
  Admin => Editor
  System Administrator => Manager


Sichtbare Rollen:
  Arbeitsraum Ersteller


Status Offen:
  Ein Benutzer kann diesen Inhalt ansehen.
  Ein Benutzer kann diesen Inhalt löschen.

  Ein Arbeitsraum Ersteller kann neuen Inhalt hinzufügen.

  Ein System Administrator kann das gleiche wie ein Arbeitsraum Ersteller.

  Ein System Administrator kann diesen Inhalt bearbeiten.
  Ein System Administrator kann diesen Inhalt löschen.
  Ein System Administrator kann Inhalte konfigurieren.
  Ein System Administrator kann Portlets verwalten.
  Ein System Administrator kann Rollen zuweisen.
